# TpyxaTerminal
Arduino library for telnet connections
=====================
Библиотека связи по протоколу телнет для связи типа ардуино-ардуино
Применение: wifi-пульты, датчики, логгеры, экраны и т.д.

# Важно
Предназначена для ESP32, но возможно будет работать на ESP8266 и других (не тестировалось)
